## 🔧 Customization

### Change Animation Speed:
Edit animations.css and adjust duration:
```css
/* Fast: 0.3s */
/* Normal: 0.6s */
/* Slow: 1.0s */
```

### Change Colors:
Replace color codes:
```css
#667eea → Your Primary Color
#3498db → Your Secondary Color
#27ae60 → Your Success Color
```

### Disable Animations:
Add to CSS:
```css
* {
    animation: none !important;
    transition: none !important;
}
```
